# 藥品比價小精靈 - 跨機移植指南 (Migration Guide)

如果您想將這個專案搬移到另一台電腦使用，請按照以下步驟進行。

## 🧰 事前準備 (一輩子只需做一次)

在新的電腦上，請確保已安裝以下軟體：
1. **Node.js** (建議版本 20.x 或以上)
   - 下載連結: [https://nodejs.org/](https://nodejs.org/)
   - 安裝時請一路點擊「Next (下一步)」即可。

## 🚀 三步快速啟動

1. **複製資料夾**：
   - 將整個 `drug-price-compare` 資料夾複製並貼上到新電腦的任何位置 (桌面、隨身碟或 D 槽皆可)。
   - **重要**：請確保資料夾內包含 `all1_11504_1.TXT` 等原始健保資料檔。

2. **執行安裝環境**：
   - 對著資料夾內的 **`setup_environment.bat`** 點擊兩下。
   - 它會自動幫您安裝必要的插件，並重新建立藥品索引。看到「安裝完成」提示後即可關閉。

3. **啟動搜尋引擎**：
   - 在資料夾內開啟終端機 (PowerShell)，輸入：
     ```bash
     npm run dev
     ```
   - 您的比價精靈就會在新電腦上華麗登場了！

---

## 🛠️ 常見問題
*   **畫面變白了？**：請檢查 `src/assets/nhi_index.json` 是否已生成，或者是重新執行一次 `setup_environment.bat`。
*   **找不到原始檔？**：請確保 TXT 檔的檔名為 `all1_11504_1.TXT` 且放在專案根目錄下。

**祝您在新環境同樣查價順利！💎**
